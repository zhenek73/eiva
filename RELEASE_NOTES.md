# Eiva v1.0.0 - Production Release

This is the first production-ready release of Eiva, an AI digital twin platform on TON blockchain.

## Release Title
**Eiva v1.0.0 — Your AI Self, Owned by You**

## 🎉 What's Included

### Core Features
- ✅ Personality extraction from Telegram chat history
- ✅ Multi-source learning (combine multiple exports)
- ✅ Personal/Professional mode switching
- ✅ RAG-based memory system with categorization
- ✅ Multi-language support (English/Russian)
- ✅ TON blockchain integration (Soul Certificate NFT)
- ✅ Web dashboard with TON Connect auth
- ✅ Production-ready code

### Advanced Features
- ✅ Memory Types (Memories/Opinions/Facts)
- ✅ Personality confidence scoring
- ✅ Hallucination control settings
- ✅ Multi-personality mode support
- ✅ Feedback collection system

## 📊 Honest About Limitations

This release includes full transparency about AI hallucinations:
- Confidence scoring for personality traits
- User-configurable hallucination mitigation
- Clear documentation of limitations
- Recommendations for improving accuracy

## 🚀 Getting Started

```bash
git clone https://github.com/zhenek73/eiva
cd eiva/eiva-bot
cp .env.example .env
py run.bat
```

Then open Telegram: [@eivatonbot](https://t.me/eivatonbot)

## 🐛 Known Limitations

- TON testnet only (mainnet coming soon)
- Rate limits on API calls
- Personality extraction works best with 50+ messages per source
- AI hallucinations possible (see /profile for confidence scores)
