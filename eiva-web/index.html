🔒 Privacy Protection

Your twin never shares your sensitive data — even if someone asks directly.

🚫 Passwords & PINs
🚫 Email addresses

🚫 Passport / ID data
🚫 API keys & tokens

How we protect your data:

Layer 1
Parser Filter

Before indexing, messages containing emails, phone numbers, passwords, card numbers are automatically removed. They never enter the AI's memory.

Layer 2
AI Instruction Lock

A hardcoded system prompt rule (non-overridable) tells the AI to refuse any request for sensitive personal data, regardless of how the question is phrased.

Layer 3
Local Storage Only

Your messages stay on your machine. Only embeddings (numeric vectors) are stored. Raw text is processed locally and never sent to external servers in bulk.
